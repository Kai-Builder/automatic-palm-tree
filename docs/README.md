# Requirements

- Docker `18.x`
- Node `8.x`

# Install

```
git clone git@github.com:rodoabad/automatic-palm-tree.git
docker-compose build
```

# Usage

```
docker-compose up
```

```
docker-compose run dev npm run test
```
